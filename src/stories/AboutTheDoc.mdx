## About the Documentation

This page explains how to read the Lovely Bases documentation.

---

### Using the documentation

- Browse individual view pages to explore layouts, examples, and configuration options.
- Use the status badge on the top right to assess whether a view is suitable for your needs.
- Testing and experimental views are best used for evaluation.
- Ready and stable views are recommended for long-term or production setups.

If something is unclear or missing, contributions and feedback are welcome.

### Status Badges

Each view includes a status badge that represents its current state. These statuses reflect how stable a view is and how likely it is to change over time.
